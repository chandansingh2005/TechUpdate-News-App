# 🚀 TechUpdate - Tech News Web App

A dynamic web application that displays real-time IT, AI, and developer-related news using an external API.

---

## 🔥 Features

- 🔍 Search functionality (AI, React, Jobs, Skills)
- 📰 Displays top 10 latest tech news articles
- 🖼️ Handles missing/broken images with fallback
- ⚡ Fast and dynamic UI updates
- 📱 Responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- News API  

---

## 📌 How It Works

1. User enters a topic (e.g., AI, Web Development)
2. App sends request to API
3. Fetches real-time news data
4. Displays articles with:
   - Image  
   - Title  
   - "Read More" link  

---

## ⚠️ Note

Due to limitations of the free API plan, the deployed version may not display news data.  
However, the application works correctly in the local development environment.
