# TechUpdate - News App 🚀

A dynamic web application that displays real-time tech news using API.

## 🔥 Features
- Fetches latest IT, AI, and developer news
- Search functionality
- Responsive design
- Error handling for missing data
- Dynamic UI updates

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- News API

## 📌 How it works
- User enters a topic (AI, React, Jobs)
- App fetches real-time news from API
- Displays top 10 articles with image and link

## 🌍 Future Improvements
- Category filtering
- Dark mode
- React version
